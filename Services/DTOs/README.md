## NOTAS:
# DTOs
Este directorio contiene los Data Transfer Objects (DTOs) utilizados en el servicio API para la gestión de menús diarios en el restaurante. Los DTOs facilitan la transferencia de datos entre las diferentes capas de la aplicación, asegurando una comunicación eficiente y estructurada.
## DTOs
# BebidaDTO.cs
El archivo BebidaDTO.cs define el Data Transfer Object para las bebidas del restaurante. Este DTO encapsula la información relevante de una bebida, facilitando su transferencia entre la capa de servicio y otras capas de la aplicación.
# ComboDTO.cs
El archivo ComboDTO.cs define el Data Transfer Object para los combos de menú ofrecidos por el restaurante. Este DTO encapsula la información relevante de un combo, facilitando su transferencia entre la capa de servicio y otras capas de la aplicación.
# MenuDiarioDTO.cs
El archivo MenuDiarioDTO.cs define el Data Transfer Object para los menús diarios del restaurante. Este DTO encapsula la información relevante de un menú diario, facilitando su transferencia entre la capa de servicio y otras capas de la aplicación.
# PlatoPrincipalDTO.cs
El archivo PlatoPrincipalDTO.cs define el Data Transfer Object para los platos principales del restaurante. Este DTO encapsula la información relevante de un plato principal, facilitando su transferencia entre la capa de servicio y otras capas de la aplicación.
# PostreDTO.cs
El archivo PostreDTO.cs define el Data Transfer Object para los postres del restaurante. Este DTO encapsula la información relevante de un postre, facilitando su transferencia entre la capa de servicio y otras capas de la aplicación.
